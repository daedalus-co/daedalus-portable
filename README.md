# Daedalus Portable Assistant

A standalone, premium AI assistant powered by **WebGPU**. Use it directly in your browser without any API keys or backend servers!

## 🚀 Features
- **Privacy First**: Everything runs on your own GPU. No cloud, no data sharing.
- **Zero Cost**: Host it for free on GitHub Pages.
- **Fast**: Leverages WebGPU for high-performance inference.

## 🛠️ How to Use
1. Open `portable_assistant.html` in a modern browser (Chrome/Edge).
2. Wait for the model (Llama-3-1B) to download into your browser cache (approx. 1GB).
3. Start chatting!

## 🌐 Hosting on GitHub Pages
1. Upload this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and click **Save**.
4. Your bot is live at `https://<your-username>.github.io/daedalus-portable/portable_assistant.html`.
